# Gamedev.js Jam 2026 - Fufu Adventures

Welcome to the official repository for our entry in the **Gamedev.js Jam 2026**.

This game was developed by a team of three people who had never created anything like a game before. It has been an incredibly fun experience, filled with learning moments and challenges that pushed us out of our comfort zones.
